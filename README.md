# cubo-desafio-landing

Landing Page para um projeto de Mestrado — Puzzle Board 3D.

## Estrutura

- `index.html` — Página principal
- `css/styles.css` — Estilos (Material Design 3, tema dark)
- `js/main.js` — Interações (scroll, reveal, formulário de lead)

## Como usar

1. Clone o repositório
2. Abra `index.html` no navegador
